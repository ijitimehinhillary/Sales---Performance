# Sales Performance Analysis

##  Project Overview
This project presents a comprehensive analysis of a sample sales dataset using Microsoft Excel. The goal is to transform raw sales data into actionable insights through data cleaning, transformation, and interactive dashboard creation.

The analysis focuses on understanding sales performance, profitability, and trends across different regions, segments, and product categories.

##  Research Questions
The analysis was guided by the following key questions:

- What differences exist in total sales across the various regions? 
- To what extent does profitability vary across product categories? 
- How is total sales distributed among different customer segments? 
- What is the overall distribution between profit and loss within the dataset? 
- How do sales patterns evolve over time? 
- How do total sales vary across product categories and customer segments?

##  Objectives
- Analyze overall sales performance  
- Identify profitable and loss-making transactions  
- Evaluate regional, product, and segment performance  
- Visualize sales trends over time  
- Build an interactive dashboard for decision-making  

##  Tools & Techniques Used
- Microsoft Excel  
- Data Cleaning & Preparation  
- Pivot Tables  
- Charts (Bar, Line, Pie)  
- Slicers (Region, Segment)  
- IF Function for classification  

##  Data Cleaning & Preparation
The dataset was imported into Excel and cleaned to ensure accuracy and consistency:
- Corrected data alignment issues  
- Standardized date formats  
- Handled missing and inconsistent values  
- Verified numerical fields (Sales, Profit)

##  Data Transformation
A new column, **Performance**, was created to classify transactions as either *Profit* or *Loss* using the IF function:

=IF(Profit>0,"Profit","Loss")

This enabled clearer analysis of business performance.

##  Data Analysis
A total of **six (6) pivot tables** were created to explore the dataset:

1. **Sales by Region** – to identify high-performing locations  
2. **Profit by Product** – to evaluate product profitability  
3. **Sales Trend Over Time** – to analyze patterns and seasonality  
4. **Profit vs Loss Distribution** – to assess business performance  
5. **Sales by Segment** – to understand customer segmentation impact  
6. **Profit by Region** – to compare regional profitability  

##  Dashboard Features
An interactive Excel dashboard was created to present key insights:

- Visualizations:
  - Bar chart (Sales by Region)  
  - Line chart (Sales Trend)  
  - Pie chart (Profit vs Loss)  
  - Additional charts from pivot tables  

- Interactivity:
  - Slicers for **Region** and **Segment** filtering  

The dashboard provides a clear and concise view of performance metrics for quick decision-making.

##  Key Insights
- Certain regions contribute significantly more to total sales  
- Some products generate higher profits while others lead to losses  
- Customer segments vary in their contribution to revenue  
- Sales trends reveal fluctuations over time, indicating possible patterns  
- Some regions show strong sales but lower profitability  

##  Conclusion
This project demonstrates the use of Microsoft Excel as a powerful tool for data analysis and visualization. It highlights how structured analysis and dashboards can support data-driven decision-making.
